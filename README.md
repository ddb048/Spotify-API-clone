# AMPLIFY

#SECTION - User Stories
As an unregistered guest to AMPLIFY, I can register an account to become a User.

As a User, I can login/logout to interact with the full application

As a User, I can view all available features organized in an easy to follow structure based on my tastes and specificity
    From least specific to most:
    A user can select collections or PLAYLISTS from a tile and listen to them.
    A user can select artists from a tile and listen to them.
    A user can select an album from a tile and listen to them.
    A user can select a track from a tile and listen to it.
